中国社会各阶级的分析的笔记

0x00 基本概念认知
首先要分清，左翼，右翼，也即是左派和右派。
#概念源自于法国大革命时期，在制宪会议上，当时来自第三阶级（市民和农民）的自由派参议员坐在主席的左侧，而第二阶级（贵族阶级）的成员则坐在右侧。
#之后的立法会，将坐在右侧，支持旧制度的君主主义者称为右派。左派则反之。

左翼和右翼在政治传统上指一个社会内部政治领域的两种意识形态，尤其是在民主社会。
在现代西方国家，政治通常被分为右翼或左翼，即保守主义（右翼）和社会主义（左翼）。
在美国，广义上的自由主义指左翼政治，而在欧洲，自由主义则指更广泛意义上的右翼政治。

##那么放在我国是怎么划分的，大概这么划分
“左”派：激进派，忽视客观条件，客观规律而急于追求成果的一种做法。”左”倾错误即例子。
左派：进步派，相信人的能动性和主体性，根据客观规律实现目标，打破阶级隔离
中间派：墙头草，两边倒
右派：保守派，维护封建主义，限制人的能动性，与左派恰恰相反

0x01 对阶级的分析的目的
教员提到:”谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题”
也即是说，我们的核心问题是分清谁是敌人，谁是朋友。
如果不进行分清，我们没法团结真正的朋友而攻击真正的敌人，消弱了革命力量。
而为了分清敌友，我们就需要对社会各阶级的地位及其经济进行分析。

0x02 对各阶级具体分析
#地主阶级和买办阶级
首先理解买办阶级：
买办阶级出现在近代鸦片战争之后，是联系中国与外国商人之间的桥梁，是帝国主义剥削和压迫本国人民的工具和奴才，买办的实质是外国侵略中国经济的工具，阻碍了中国工商业的发展。
地主阶级和买办阶级都是国际资产阶级的附庸，是他们的走狗。他们代表着中国最落后的生产关系，阻碍中国生产力发展。
是极端的反革命派，是我们的敌人。

#中产阶级
处于一种矛盾：被外国资产阶级和军阀打压，他们需要革命；但需要跻身大资产阶级时，又怀疑革命。
这样的矛盾最终之后全部破灭，他们要么向左加入革命，要么向右反对革命。
他们想要实现民族资产阶级统治我们的国家，这是不行的。
其左翼可能是我们的朋友，其右翼可能是我们的敌人。

#小资产阶级
同样分成了左派右派
右派：能自给自足，劳动量没有加大，他们的资产接近中产，因此也对中产阶级的态度将信将疑，他们害怕革命，但是是少数派。
左派：加大劳动量之后能够自给自足的，以及感到生活质量下降的。他们一样痛恨洋鬼子，他们也会加入革命。
当左派大多数进入了革命的潮流，右派也只得跟着进入，我们要争取他们为我们的朋友

#半无产阶级
半自耕农，店员，小手工业者等，劳累的工作下能够供家庭的费用，但也是叫苦不迭，需要一个革命来改善他们的地位。

#无产阶级
工人，贫农，失业者。是我们的中坚力量，我们要靠工人的力量进行革命

0x03 结论
无产阶级是我们的革命领导力量，我们要争取半无产阶级和小资产阶级，对其正确的引导，成为我们的战友；对中产阶级多家提防；坚决打击地主阶级和买办阶级

0x04 思考以及后记
现如今的社会好像开始走偏，甚至能在一些地方看到了买办阶级的影子，甚至资产阶级的影子也出现了。我想这是不好的，我们是坚定的社会主义者。
我们的朋友是工农阶级，而在如今的社会下，半无产阶级的数量大大提升，他们被上头压迫剥削劳动力，只为了能养活自己或者供给家庭，他们也是我们的朋友。
为了打破阶级的对立，为了反抗压迫和剥削，我们依然需要认清局势，团结我们的一切朋友，坚决对抗资产阶级和买办阶级。

    blacklist = ['config', 'self']
    return ''.join(['{{% set {}=None%}}'.format(c) for c in blacklist]) + s
